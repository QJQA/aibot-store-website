Convert files directly in your Discord server. Simple, fast, efficient.

Add to Discord

How to Use

  1. Add bot to your server
  2. Upload file to Discord
  3. Type !convert [format]
  4. Receive converted file

Example: To convert a file to PDF, upload the file and type:

!convert pdf

Commands

  • !formats - List supported formats
  • !help - Show help
